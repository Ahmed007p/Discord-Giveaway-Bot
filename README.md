 Commands

/giveaway start

Start a new giveaway

路 duration - Time duration (1h, 2d, 30m)
路 winners - Number of winners
路 prize - Giveaway prize
路 description - Requirements (optional)

/giveaway end

End a giveaway manually

路 message_id - Giveaway message ID

/giveaway reroll

Re-select winners

路 message_id - Giveaway message ID
路 winners - Number of new winners
